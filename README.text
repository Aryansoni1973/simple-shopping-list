Description
This is a simple web application for managing your shopping list. You can easily add items to your list and keep track of what you need to buy.

Table of Contents
1. Installation
2. Usage
3. Contributing
4. License
5. Contact

Installation
Clone this repository to your local machine:
    git clone <https://github.com/Aryansoni1973/simple-shopping-list>
Navigate to the project directory:
    cd shoppingList
Open index.html in your web browser.

Usage
1. Enter the item you want to add to your shopping list in the input field provided.
2. Click on the "Add" button to add the item to your list.
3. The item will be displayed in the item container below the input field.
4. Repeat steps 1-3 to add more items to your list.
5. To remove an item, click on the item in the list.
